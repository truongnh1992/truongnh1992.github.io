### How to build this blog?

#### 1. Install Bundler
```console
gem install bundler
bundle update
```

#### 2. Clone source code

```console
git clone https://github.com/truongnh1992/truongnh1992.github.io.git
```

#### 3. Run the blog on localhost

```console
truongnh@k8s:~/truongnh1992.github.io$ bundle exec jekyll serve
```
